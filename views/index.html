These4Words Map

Remember a location anywhere in the world with just four words.

This is hot off the press. We are still finalising the word list so locations might not be reproducible. So do not use it to mark the location of your antarctic resupply drop quite yet!

  • Type in a location to find out the four words by which to remember it (try 1600 Pennsylvania avenue ...)
  • To find the location referenced by a set of four words given to you, type them in separated by a dash (try {{ sydney }})
  • Go straight to exploring the map by clicking the Find on Map button, it will take you to Sydney

Some interesting locations to try out:

  • Battery Park, NYC
  • Downtown San Francisco
  • Sydney, Australia
